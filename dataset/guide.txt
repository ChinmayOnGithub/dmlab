üöÄ EXAM STRATEGY:
================

STEP 1: IDENTIFY DATA TYPE (2 minutes)
- Look at target column
- Check data format (numeric/categorical/transactions)

STEP 2: CHOOSE METHOD (1 minute)
- Classification: Categorical target
- Prediction: Numeric target
- Clustering: No clear target
- Association: Transaction data

STEP 3: ATTRIBUTE SELECTION (5 minutes)
- Remove irrelevant columns (ID, Name, Phone)
- Keep features that affect target
- Justify your choices

STEP 4: DATA EXPLORATION (10 minutes)
- Show basic statistics (min, max, mean)
- Count categories
- Identify patterns

STEP 5: PREPROCESSING (15 minutes)
- Handle missing values
- Normalize if needed
- Convert data types if required

STEP 6: APPLY ALGORITHM (3+ hours)
- Implement chosen method
- Show intermediate steps
- Display results clearly

STEP 7: CONCLUSION (5 minutes)
- Interpret results
- Comment on accuracy/effectiveness
- Suggest improvements
